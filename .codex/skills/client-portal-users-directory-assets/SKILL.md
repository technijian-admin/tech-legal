# Client Portal Users, Directory & Assets

Use this skill for user and directory-oriented data.

## Workflow

1. Search with `user`, `directory`, `client users`, `location users`, or `asset` terms.
2. Read [references/coverage.md](references/coverage.md).

4. Redact secrets unless the user explicitly asks for them and the request is appropriate.
