# Client Portal Time Entries & Approvals

Use this skill for labor tracking and approval workflows.

## Workflow

1. Search with `time entry`, `approve time`, `workshift`, `pod`, or `eom` terms.
2. Read [references/coverage.md](references/coverage.md).
3. Prefer read/list procedures first; many time-entry saves are write-sensitive.
4. When a time-entry query returns nothing, verify the ticket ID, date window, or approval state before concluding there is no data.

## First Search Phrases

- `time entry ticket`
- `approve time entry`
- `workshift`
- `pod user workshift`
- `eom approval`
