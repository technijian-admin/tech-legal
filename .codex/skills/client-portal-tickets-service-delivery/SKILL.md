# Client Portal Tickets & Service Delivery

Use this skill for ticket-centered operational work.

## Workflow

1. Search with `ticket`, `status`, `client`, `attachment`, or `recurring` terms.
2. Read [references/coverage.md](references/coverage.md).
3. Default to read-only ticket views unless the user explicitly asks to create or update a ticket.
4. Keep attachment and meeting lookups close to the ticket request they support.

## First Search Phrases

- `ticket detail`
- `ticket client`
- `ticket status`
- `ticket version`
- `ticket attachment`
- `recurring ticket history`
