# Client Portal Invoices, Billing & Payments

Use this skill for revenue and payment workflows.

## Workflow

1. Search with `invoice`, `weekly invoice`, `monthly invoice`, `proposal invoice`, `cardpointe`, or `quickbook`.
2. Read [references/coverage.md](references/coverage.md).
3. Distinguish read/report procedures from generate/create procedures before executing.
4. If the user asks to create or regenerate invoices, pause and confirm intent in your response because many billing procedures write data.

## First Search Phrases

- `clients invoices`
- `weekly invoice`
- `monthly invoice`
- `proposal invoice`
- `invoice preview`
- `cardpointe profile`
