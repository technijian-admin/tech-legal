QuickBooks Enterprise Direct-SDK accounting service (.NET 8, `net8.0-windows`, x86). Phase 1 is the skeleton: the `IRequestProcessor` COM seam (`QbConnectService.Qb` namespace, living in the isolated `QbConnectService.Qb.Com` project so the host and tests stay SDK-free), `FakeRequestProcessor` in tests, a Phase-1-stub `RealRequestProcessor` whose real COM adapter arrives in Phase 2, a tri-mode host (`AddWindowsService()` so the same exe runs as a console app, Windows service, or scheduled task), and CI in `.github/workflows/quickbooks-ci.yml`. The install, uninstall, certificate, task-registration scripts, and the full `10.120.254.13` deploy runbook arrive in Phase 9. Design spec: `docs/superpowers/specs/2026-05-11-quickbooks-direct-sdk-accounting-design.md`; roadmap: `.planning/ROADMAP.md`.
