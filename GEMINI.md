# Gemini Guide

Read [`AGENTS.md`](./AGENTS.md) first.

For Client Portal work in this repo:

1. Use the shared playbook under `.codex/skills/`.
2. Start with `.codex/skills/client-portal-core/SKILL.md`.
3. Use `.codex/skills/client-portal-core/references/domain-map.md` to select the right domain skill.
4. Use `.codex/skills/client-portal-core/scripts/client_portal_api.py` for live API search, guide lookup, and execution.

The `.codex/skills/` directory is Codex-formatted, but the files are also intended to be readable by other agents as repo-local operating instructions.
