## Drafting Notes (delete before issuance)

1. **Verify the request reason.** Check the requesting box and capture the recipient (bank, embassy, etc.) for audit trail.
2. **Cap the validity.** This certificate should be treated as valid for **30 days** from the date of issuance for third-party verification purposes. Add a footer if the recipient requires a specific expiry.
3. **Two versions exist:** (a) **Annual CTC version** (this template) for loan/visa/lease — gives the full structure; (b) **Last-3-Months Payslip version** for some banks — issue alongside actual payslips.
4. **Confidentiality reminder.** Include the §"Verification" paragraph above so HR doesn't leak salary data to cold-callers.
5. **Do NOT issue retrospectively to ex-employees** — for ex-employees, use the [`Service-Letter.md`](../07-Separation/Experience-Letter.md) template instead, which only confirms tenure and last-drawn salary at separation.
6. **Loan/HRA/visa-specific quirks:**
   - Banks typically want last 3 months' payslips alongside this certificate
   - HRA proofs need rent receipts + landlord PAN if rent > ₹1L/year
   - US/UK/Canada visa applications often want a letter on company letterhead with employer signature + stamp + a no-objection-to-travel clause; consider drafting a separate Visa Support Letter
