# Compliance Calendar вҖ” Technijian Chandigarh + Panchkula

**Last verified:** 2026-05-06.

Date-driven compliance calendar consolidating all monthly, quarterly, half-yearly, and annual deadlines that apply across Technijian's Chandigarh and Panchkula offices.

For per-location detailed checklists see [chandigarh-checklist.md](chandigarh-checklist.md) and [panchkula-checklist.md](panchkula-checklist.md).

---

## Monthly recurring deadlines

| Day of month | Activity | Both / specific | Authority |
|---|---|---|---|
| **7th** | TDS deduction & deposit (challan) | Both | Income-tax |
| **7th** | Salary payment for previous month | Both | Code on Wages В§17 |
| **15th** | EPF вҖ” Electronic Challan-cum-Return (ECR) | Both | EPFO |
| **15th** | ESI вҖ” Monthly contribution challan | Both | ESIC |

(Day refers to the day of the month following the period to which the obligation relates вҖ” e.g., March payroll вҶ’ 7-April / 15-April.)

## Quarterly deadlines

| Quarter | Period covered | Deadline | Activity |
|---|---|---|---|
| Q1 (Apr-Jun) | 1-Apr to 30-Jun | 31-Jul | TDS Form 24Q (Q1) |
| Q2 (Jul-Sep) | 1-Jul to 30-Sep | 31-Oct | TDS Form 24Q (Q2) |
| Q3 (Oct-Dec) | 1-Oct to 31-Dec | 31-Jan | TDS Form 24Q (Q3) |
| Q4 (Jan-Mar) | 1-Jan to 31-Mar | 31-May | TDS Form 24Q (Q4) |

Apprentices вҖ” quarterly progress report: typically by 15th of the month following quarter end.

OT cumulative review (вүӨ50 hrs/quarter per employee): end of each quarter.

## Half-yearly deadlines (LWF вҖ” major difference between locations)

| Period | LWF Apr-Sep | LWF Oct-Mar |
|---|---|---|
| **Chandigarh UT (Punjab LWF)** | Deposit + Form A by **15-October** | Deposit + Form A by **15-April** |
| **Panchkula (Haryana LWF)** | Deposit + Form A by **15-October** | Deposit + Form A by **15-April** |

ESI half-yearly returns: as prescribed (typically Oct for Apr-Sep, Apr for Oct-Mar).

## Annual deadlines

| Date / period | Activity | Authority | Both / specific |
|---|---|---|---|
| **31-December (prior year)** | Publish next-calendar-year holiday list at the establishment | State Holidays Act | Both |
| **30-April / 31-May** | Income-tax return of establishment (CIT / FY 25-26 = AY 26-27) | Income-tax | Both |
| **15-June** | Issue Form 16 to all employees for previous FY | Income-tax | Both |
| **30-November** | Pay statutory bonus for previous FY (if applicable; вүӨ 8 months from FY close) | Code on Wages В§39 | Both |
| **End of January** | POSH IC Annual Report filed with District Officer (DC Panchkula / DC Chandigarh) | POSH Act | Both |
| **At AGM** | Director's / company report вҖ” POSH disclosure under Companies Act В§134(3) | Companies Act | Both |
| **As scheduled** | Refresher POSH training (вүӨ 12 months from last) | POSH Act | Both |
| **As scheduled** | Refresher POSH IC training (вүӨ 12 months) | POSH Act | Both |
| **As scheduled** | Renew Fire NOC | State Fire Services | Both |
| **As scheduled** | Renew Trade Licence | Municipal Corporation | Both |
| **Annual** | Audit CTC structure вҖ” Basic + DA вүҘ 50% | Code on Wages В§2(y) | Both |
| **Annual** | Audit contract-labour / vendor compliance | OSH Code | Both |
| **Annual** | DPDP вҖ” review HR data retention; erase non-essential data | DPDP Act | Both |
| **Apr & Oct (typical)** | Re-verify state minimum wages notification | Code on Wages | Both |
| **Annual (Panchkula only)** | Renew **Women's Night-Shift Exemption** (apply 1 month before expiry) | 8-May-2025 notification | Panchkula |
| **Annual (Apr / Oct typical)** | VDA reconciliation against state notification | Code on Wages | Both |
| **As prescribed** | EPF Annual Return (Form 6A consolidated) | EPFO | Both |

| Crossing 20 employees | EPF registration | 60 days |
| Crossing 30 employees | Apprentices Portal registration | Within reasonable period |
| Crossing 50 employees | CrГЁche facility | Before next maternity-leave applicant |
| Crossing 100 employees | Canteen | OSH Code |
| Crossing 250 employees | Welfare Officer | OSH Code |
| Crossing 300 workers | Standing Orders + Govt-permission for retrenchment | Before need |
| New employee joining | Day-1 onboarding (offer letter, EPF Form 11, ESIC Form 1, gratuity nomination, POSH, HR Privacy Notice) | Day 1 |
| Employee exit | Final settlement (2 days), gratuity (30 days), Form 16 | As prescribed |
| Address / headcount change | Update S&E registration | 15 days |
| Workplace accident | Notify Inspector + EPFO/ESIC | 24-48 hrs |
| POSH complaint received | IC inquiry initiated | 7 days; resolved 90 days |
| Personal-data breach | Notify Data Protection Board + affected employees | ~72 hours |
